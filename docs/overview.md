# 系统概览

## 架构

- 前端：静态页面，负责账号管理交互和状态展示。
- 后端：Node.js 单文件服务，提供 API、静态资源托管、数据持久化。
- 存储：本地 JSON 文件（`backend/data/accounts.json`、`backend/data/background_refresh.json`）。

## 核心模块

- 账号模块：账号增删查、授权码建号、导入导出。
- OAuth 模块：生成授权 URL、保存会话、兑换 token。
- 用量模块：调用 Claude 官方用量接口拉取账号用量。
- 后台刷新模块：服务端常驻定时任务，页面关闭后仍会继续执行。

## 页面能力

- 账号列表展示（分页、筛选、搜索）。
- 添加账号（普通方式 + OAuth 授权码方式）。
- 删除账号。
- 手动刷新与前端自动刷新。
- 后台刷新状态展示与配置。
- 数据导入与导出。

## 运行方式

```bash
node backend/server.js
```

默认监听 `127.0.0.1:8787`。
